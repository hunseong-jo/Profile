## 방학 보강 계획 (8주 로드맵)

> 💡 **원칙:** 새로 만드는 것보다 **이미 한 것을 정리하는 것**이 훨씬 빠르고 설득력 있다.

---

### PHASE 1 — 빠른 완성 (1~2주차)
> 하루 1~2시간, 당장 할 수 있는 것부터

#### ✅ 1-1. 자기소개 한 문장 작성
- **목표:** 포트폴리오 히어로 섹션의 빈 자리 채우기
- **방법:** 아래 템플릿 참고해서 본인 버전으로 작성
  ```
  "[형용사] + [직무] 지망생으로, [핵심 경험/강점]을 가진 [이름]입니다."
  
  예시:
  "게임을 분석하고 더 재미있게 만드는 것을 좋아하는 게임 기획자 지망생,
   실제 런칭 경험을 보유한 조훈성입니다."
  ```
- **소요 시간:** 2~3시간
- **효과:** 포트폴리오 완성도 즉시 상승

#### ✅ 1-2. 증명사진 촬영
- **방법:** 근처 사진관 or 스마트폰 + 흰 벽 + 자연광
- **포인트:** 밝은 표정, 단색 배경, 상반신 구도
- **소요 시간:** 반나절
- **효과:** 신뢰감 즉시 상승

#### ✅ 1-3. 쥐독한 등반 포스트모템 작성
- **목표:** 이미 만든 게임을 **기획 문서로 재정리** → 바로 포트폴리오 작업물이 됨
- **포함 내용:**
  - 게임 컨셉 한 장 요약
  - 내가 기획한 함정/맵 목록 + 의도 설명
  - 잘 된 점 / 아쉬운 점 각 3가지
  - 다음에 개선할 것
- **형식:** Notion 페이지 or PDF 1~2장
- **소요 시간:** 3~5일
- **효과:** "기획했다"는 주장에서 "기획서를 보여줄 수 있다"로 격상

---

### PHASE 2 — 기획 작업물 만들기 (3~5주차)
> 가장 효과 대비 노력이 높은 구간

#### ✅ 2-1. 게임 분석 글 2편 작성
- **목표:** 즐겨 하는 게임 2개를 기획자 시각으로 분석
- **분석 항목:**
  - 핵심 재미 요소 (Core Loop)
  - 튜토리얼 구조
  - 레벨/스테이지 난이도 곡선
  - 잘 된 점 / 아쉬운 점
  - "내가 기획자라면 어떻게 바꿀까"
- **형식:** 블로그 포스팅 (Tistory 또는 Velog 추천)
- **소요 시간:** 편당 2~3일
- **효과:** 글쓰기 능력 + 게임 분석력 동시 증명

#### ✅ 2-2. 새 게임 아이디어 기획서 작성 (페이퍼 프로토타입)
- **목표:** 아직 만들지 않은 게임을 기획서로만 완성
- **포함 내용:**
  - 타겟 플레이어 정의
  - 핵심 재미 요소 1줄 정의
  - 게임 플로우 다이어그램 (손으로 그려도 OK)
  - 레벨 구성 스케치 3개 이상
  - 조작 방식 설명
- **형식:** Notion / Google 슬라이드 / PDF
- **소요 시간:** 1~2주
- **효과:** 실제로 만들지 않아도 기획력을 증명 가능

#### ✅ 2-3. 레벨 디자인 문서 1개 제작
- **목표:** 쥐독한 등반 or 새 기획의 레벨을 시각적으로 설계
- **방법:**
  - 방눈금 종이 or Excalidraw(무료 온라인 툴) 사용
  - 플레이어 동선, 함정 위치, 체크포인트 표시
  - 각 구간의 의도 설명 (예: "여기서 속도감을 높이기 위해 ~")
- **소요 시간:** 3~5일
- **효과:** 레벨 디자이너 지망에 직결되는 핵심 작업물

---

### PHASE 3 — 온라인 존재감 구축 (6~7주차)
> 발견 가능성을 높이는 작업

#### ✅ 3-1. 블로그 개설 및 글 이식
- **플랫폼 추천:** Tistory (게임 분야 노출 좋음) 또는 Velog
- **PHASE 2에서 쓴 분석글 2편 바로 업로드**
- **소요 시간:** 반나절 (글은 이미 작성됨)

#### ✅ 3-2. GitHub 포트폴리오 정리
- **현재 상태:** contact.html + game-cover.png만 있음
- **추가할 것:**
  - README.md에 자기소개 + 프로젝트 목록 작성
  - 기획서 PDF 업로드
- **소요 시간:** 1~2일

---

### PHASE 4 — 새 프로젝트 도전 (8주차, 여유 있을 때)
> 시간이 남으면 하는 것, 완성도보다 기록이 중요

#### ✅ 4-1. 초소형 Unity 게임 1개 제작
- **규모:** 5분 안에 끝나는 간단한 게임
- **예시:** 피하기 게임, 점프 게임, 클리커 게임
- **포인트:** 완성 + STOVE 또는 itch.io 업로드
- **소요 시간:** 2~3주 (기획 1주 + 개발 1~2주)
- **효과:** 프로젝트 2개로 증가 + 개인 역량 증명
